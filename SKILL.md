# A-Share Quant Manager

A股量化交易管理系统

## 核心纪律

1. **高位放量不追，等回调** - 短期涨幅过大、放量滞涨的股票，禁止追高买入
2. **基于真实数据回答** - 所有个股、板块相关讨论，必须先调用skill获取实时数据，禁止基于记忆或推测回答

## 功能模块

### 1. 数据获取 (core/data_fetcher.py)
- 多数据源：腾讯、东方财富、新浪、AKShare
- 自动切换，互为兜底
- 只使用真实数据，禁止AI生成假数据

### 2. 自选股管理 (core/watchlist_memory.py)

- 持久化存储

### 3. 自动选股管理 (core/auto_watchlist_manager.py)
**五策略自动选股系统 + 多维度分析引擎**

#### 核心升级：多维度分析引擎 (core/multi_dimension_analyzer.py)
```
总分 = 趋势(30%) + 基本面(25%) + 资金(20%) + 技术(15%) + 板块(10%)
```

**分析维度**：
- **趋势面**：股价位置、均线排列、近期趋势
- **基本面**：估值水平(PE/PB)、盈利能力(ROE)、成长性
- **资金面**：主力流向、成交量变化、大单买入
- **技术面**：MACD状态、支撑压力、K线形态
- **板块面**：所属板块强度、板块内排名

#### 策略0: 强势股低吸 💧（核心策略）
- **逻辑**：昨天涨得好证明强势，今天回调给出买点
- **特征**：热门板块核心标的、昨天强势、今天回调1-4%
- **举例**：昨天+2.5%，今天-1.5% → 低吸机会

#### 策略1: 追涨型 🚀
- **标的**：板块龙头、强势股（量价齐升、突破新高）

#### 策略2: 潜力型 💎
- **标的**：热门板块核心标的（微涨蓄势、资金关注）

#### 策略3: 抄底型 🎯
- **标的**：调整后低位企稳的热门股（回调>10%、止跌信号）

#### 策略4: 多维度优选 ⭐（新增核心策略）
- **逻辑**：使用多维度分析引擎，综合五维度评分
- **入选条件**：综合评分≥65分
- **特点**：不只看涨跌，而是全面评估趋势、基本面、资金、技术、板块
- **标记**：⭐重点监控-优选

#### 策略5: 震荡整理 ⏳
- **逻辑**：评分60-64分中等偏上股，今日涨幅-1%~+1%震荡
- **特征**：震荡整理中，等待突破信号
- **适用**： breakout交易，突破后追涨

#### 策略6: 板块补涨 📈
- **逻辑**：板块涨但个股没涨，补涨机会
- **特征**：板块涨幅>1.5%，个股涨幅<0.5%
- **输出**：板块涨幅、个股涨幅、补涨潜力评分

#### 策略7: 基本面价值 💰
- **逻辑**：PE<20 + ROE>15%，价值投资视角
- **特征**：低估值+高盈利，不管今日涨跌
- **适用**：中长期持有，防御型配置

#### 策略8: 重点板块筛选 🔍
- **逻辑**：指定重点关注板块，深度扫描优质标的
- **默认板块**：AI算力、半导体、新能源、机器人
- **输出**：每个板块前3名，按评分排序

**自动管理逻辑**:
- 每30分钟扫描五种策略标的
- 强势股低吸 + 多维度优选 双核心策略
- 不同策略不同止损条件
- 综合评分优先于单一指标

### 4. 板块跟踪 (core/sector_tracker.py)
**板块轮动监控 - 支持动态板块管理**
- 扫描7大默认板块（AI算力、半导体、新能源、医药、金融、消费、机器人）
- **动态板块**：支持添加新板块概念（如：固态电池、低空经济等）
- 自动识别板块龙头
- 自动加入板块龙头到自选（🏆标记）

**动态板块管理API**:
```python
from core.sector_tracker import get_sector_tracker

sector = get_sector_tracker()
# 添加新板块
sector.add_sector('固态电池', ['300073', '002709', '603659'])
# 删除板块
sector.remove_sector('旧板块')
# 向板块添加个股
sector.add_stock_to_sector('AI算力', '300418')
```

## 动态板块管理

系统支持动态添加新板块概念，无需修改代码：

### 方法1：通过API添加
```python
from core.sector_tracker import get_sector_tracker

sector = get_sector_tracker()
# 添加新板块概念
sector.add_sector('固态电池', ['300073', '002709', '603659', '300450'])
sector.add_sector('低空经济', ['002151', '300900', '002097', '600038'])
```

### 方法2：通过配置文件
编辑 `data/sectors.json` 文件：
```json
{
  "固态电池": ["300073", "002709", "603659"],
  "低空经济": ["002151", "300900", "002097"]
}
```

### 板块管理命令
- `add_sector(name, codes)` - 添加新板块
- `remove_sector(name)` - 删除板块
- `add_stock_to_sector(sector, code)` - 向板块添加个股
- `remove_stock_from_sector(sector, code)` - 从板块移除个股
- `get_all_sectors()` - 获取所有板块名称
- `get_sector_stocks(sector)` - 获取板块成分股

## 自选池自动管理机制

### 自动加入条件
1. **买入信号股**：评分≥50分（量价齐升+强势上涨+突破新高）
2. **板块龙头**：所属板块涨幅前3 + 个股领涨（涨幅>3%）

### 自动降级条件
1. **形态走坏**：重点监控股票大跌（-5%以上）
2. **跌破支撑**：关键支撑位失守

### 分类体系
- **重点监控** ✳️：买入信号股，priority=10，高频监控
- **板块龙头** 🏆：板块领涨股，priority=8，中频监控
- **观察**：普通关注股，priority=0，低频监控

### 买入建议输出
- 🚀 强势买入信号（涨幅>3%）
- ⏳ 观察等待（涨幅0-3%）
- ❌ 取消关注（跌幅>5%）
